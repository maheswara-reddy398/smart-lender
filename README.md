# Smart Lender - Loan Approval Prediction System

## Project Overview

Smart Lender is a Machine Learning-based web application that predicts whether a loan application is likely to be approved based on applicant details.

The application is built using Python, Flask, Scikit-learn, and XGBoost.

---

## Features

- Data Visualization
- Data Preprocessing
- SMOTE for Data Balancing
- Feature Scaling
- Multiple Machine Learning Models
- Random Forest Model
- Flask Web Application
- Loan Approval Prediction

## How to Run

1. Clone the repository

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Flask application

```
cd Flask
python app.py
```

4. Open your browser

```
http://127.0.0.1:5000
```
